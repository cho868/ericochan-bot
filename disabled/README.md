# 眠っている機能（disabled）

元の `server.js` にコメントアウトされて入っていた「今は動いていない機能」を、
復活候補としてここに集めました。`legacy-features.js` に元コードを参考として残しています。

> ここのファイルは **どこからも読み込まれていません**（＝動きません）。
> 復活させたい時は、該当ロジックを `src/features/` 配下のモジュールに移植し、
> `src/events/messageCreate.js` などから呼び出してください。
> ハードコードされたID類は `config.js`（.env）に移すのを忘れずに。
