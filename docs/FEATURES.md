# エリコちゃんbot 機能一覧（全機能カタログ）

`server.js` を解析して、現在の搭載機能を棚卸ししたものです。
「✅ 稼働中」＝今そのまま動く機能、「💤 停止中」＝コード内にあるがコメントアウト等で動いていない機能。

> ⚠️ 重要：チャンネル・ユーザー・ロール・絵文字のIDは `.env` で管理しています
> （[IDS.md](./IDS.md) 参照）。値は **元の Discord サーバー（IKKIHOUSE）専用**なので、
> 別サーバーで動かす場合は `.env` のIDを差し替えてください。

### コマンド（`!` ではじめる）

| コマンド | 説明 |
| --- | --- |
| `!help` | プリコネ用コマンドの説明Embedを表示 |
| `!クリティカル [クリ値]` | 同レベル相手へのクリティカル率を計算＋「ソルコンくじ」の運試しEmbed |
| `!持ち越し [秒数] [TL...]` | 持ち越し秒数でタイムライン(TL)を再計算して表示 |
| `!ばんぴっく` | スマブラ等のステージBAN/PICKを、リアクション（1️⃣2️⃣3️⃣4️⃣→8️⃣9️⃣）で順番に進行 |
| `!tabelogembed` | 「食の倉庫」カテゴリーの説明Embedを表示（元メッセージは削除） |
| 上記以外の `!◯◯` | 「そのコマンドはないですね… !helpで使えるコマンドが見れますよ」と返答 |

## 移行時の注意（新しいサーバー／VPSで動かす場合）

1. **IDは .env に集約済み**
   チャンネル・ユーザー・ロール・絵文字のIDは `.env`（[IDS.md](./IDS.md)）にまとめてあります。
   同じサーバーで動かす限りはそのままでOK。別サーバーに引っ越すなら `.env` の値を
   新サーバーのものに置き換えるだけで済みます（ソースコードの修正は不要）。

2. **LINE連携（GAS_URL）に注意**
   「本スレ」チャンネルでの発言は GAS へ送信されます。`GAS_URL` が未設定・無効だと、
   送信失敗のたびに「更新に失敗しました」と返信してしまう可能性があります。
   LINE連携を使わないなら、この機能を無効化しておくのが安全です（対応可能、相談ください）。

3. **Puppeteer（温泉URL収集）が重い**
   URLが貼られるたびにヘッドレスChromeを起動します。メモリの少ない無料VPSでは
   負荷・メモリ不足の原因になりやすい箇所です。

4. **discord.js v13 / Node 16.6+**
   現在のコードは discord.js v13 前提です。`npm audit fix --force` で
   むやみに更新するとv14系になり動かなくなるので注意。
