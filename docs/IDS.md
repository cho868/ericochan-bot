# ID対応表

`server.js` に直書きされていた Discord のID類は、すべて `.env`（環境変数）に移しました。
このファイルは「どのキーが何を指すか」のカタログです。

> ⚠️ 実際のID値はここには載せていません（リポジトリに残さない方針）。
> 実値は各自の `.env` に記入してください。`.env` は Git管理外です。
> ※ チャンネルID等は秘密情報ではありませんが、整理のため外出ししています。

## 認証

| キー | 意味 |
| --- | --- |
| `DISCORD_BOT_TOKEN` | Botのトークン（必須・これだけは厳重に秘密） |
| `GAS_URL` | LINE連携用 Google Apps Script のURL（任意） |
| `CRON_KEY` | cron用エンドポイントの合言葉（後述） |

## チャンネル

| キー | 意味 | 使われている機能 |
| --- | --- | --- |
| `CH_MAIN` | 本スレ | GAS連携（本スレ発言の転送） |
| `CH_INFO` | ボット通知 | お目覚めメッセージ・通話開始/終了通知 |
| `CH_DEVOJAPAN` | devojapan | （眠っている機能で使用） |
| `CH_INFORMATION` | インフォメーション | `!更新` |
| `CH_INFORMATION_CHECK` | インフォメーション確認用 | `!更新` |
| `CH_ONSEN` | 湯の倉庫 | 温泉URL自動収集 |
| `VC_MAIN` | 総合音声通話（ボイス） | 通話開始/終了通知 |
| `CH_TABELOG_KANSAI` | 食べログ・関西 | 食べログ自動仕分け |
| `CH_TABELOG_KANTO` | 食べログ・関東 | 同上 |
| `CH_TABELOG_TOHOKU` | 食べログ・東北/北海道 | 同上 |
| `CH_TABELOG_HOKURIKU` | 食べログ・北陸/東海 | 同上 |
| `CH_TABELOG_CHUSHIKOKU` | 食べログ・四国/中国 | 同上 |
| `CH_TABELOG_KYUSHU` | 食べログ・九州/沖縄 | 同上 |

## ユーザー・ロール

| キー | 意味 | 使われている機能 |
| --- | --- | --- |

| `ROLE_VC_ANNOUNCE` | 通話開演アナウンスのメンション先ロール | 通話開始通知 |

## 補足：インフォメーションEmbed内のチャンネル参照

`!更新` で表示するインフォメーションEmbedの本文には、`<#チャンネルID>` 形式の
リンク（はじめての人向けチャンネル等）が**文章として**埋め込まれています。
これは表示テキストの一部なので `.env` には外出しせず `src/features/information.js` に
そのまま残しています。文面を変えたい時はそのファイルを編集してください。
